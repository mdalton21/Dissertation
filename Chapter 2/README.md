# Chapter 2

This folder contains replication materials for Chapter 2 (Electoral Violence Networks).

## Structure
- `scripts/data_cleaning.R`: prepares network-ready ECAV actor-target data.
- `scripts/data_analysis.R`: builds network objects, generates figures, and exports tables.
- `data/`: input datasets.
- `figures/`: generated chapter figures.

## Required public datasets (not included)
- ECAV source extract (`ecav.xls` or `ecav_main.csv` depending on script use)

Place source datasets in `Chapter 2/data/`.
