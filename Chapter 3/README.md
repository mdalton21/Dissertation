# Chapter 3

This folder contains replication materials for Chapter 3 (Gender Quotas and Electoral Violence).

## Structure
- `scripts/data_cleaning.R`: prepares ECAV/ACLED analysis files and descriptive figures.
- `scripts/data_analysis.R`: estimates main models and exports model figures/tables.
- `scripts/supplement_analysis.R`: supplementary models and robustness checks.
- `data/`: chapter data inputs and derived files.
- `figures/`: generated figures.
- `tables/`: generated LaTeX tables.

## Required public datasets (not included)
- V-Dem
- ACLED
- QoG
- ECAV source file

Place public source files in `Chapter 3/data/` with filenames expected by scripts.
